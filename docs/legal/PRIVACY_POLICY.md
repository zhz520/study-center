# 隐私政策 (Privacy Policy)

**发布日期：2026年02月24日**
**生效日期：2026年02月24日**

### 1. 我们如何收集您的个人信息

1.1 **账号信息：** 注册与登录时收集的用户名、密码（加密存储）。
1.2 **社交账号同步：** 绑定 QQ 后，我们会经您授权获取您的 **QQ OpenID、昵称与头像 URL**。这仅用于展示您的个人档案及实现一键登录。
1.3 **学习数据：** 收集您的刷题进度、正确率、题目收藏状态，以便多端同步。
1.4 **设备信息：** 为保障系统运行稳定和防止恶意攻击，我们会收集设备型号、系统版本等基础硬件描述信息。

### 2. 我们如何使用您的个人信息

- 提供核心的学习、刷题及数据备份服务。
- 利用 AI 模块为您生成定制化的学情分析报告。
- 保障账号安全，验证登录状态（通过 Token 机制）。

### 3. 您的个人信息如何存储

- **存储期限：** 只要您保留账户，我们就会持续保存您的信息。
- **存储地域：** 个人信息存储在中国境内云端服务器中。
- **安全保障：** 采用行业标准的加密技术（HTTPS, 哈希算法）确保传输与存储安全。

### 4. 信息共享与转让

详见[《第三方信息共享清单》](./THIRD_PARTY_SHARING.md)。我们承诺不会主动将您的信息出售给商业机构。

### 5. 您的权利

- 您可以随时在 App 内修改密码或删除账号下的所有做题数据。
- 您可以随时解除 QQ 账号的绑定。
- 您可以通过系统设置拒绝不必要的权限申请。

### 6. 注销与删除

在「数据中心」-「危险区域」中，您可以执行“彻底清空数据”操作，我们将立即从服务器端物理删除您的所有敏感记录。
