# 第三方信息共享清单

为了向您提供更完善的服务，我们的 App 中集成了一些第三方 SDK。下表列出了我们接入的第三方 SDK 名称、使用目的、收集的信息类型以及相关的隐私政策链接。

| 第三方名称             | SDK 名称            | 使用目的                                   | 收集的个人信息类型                                       | 隐私政策/官网链接                                  |
| :--------------------- | :------------------ | :----------------------------------------- | :------------------------------------------------------- | :------------------------------------------------- |
| **腾讯科技**           | **QQ 开放平台 SDK** | 支持 QQ 一键绑定与登录、获取 QQ 昵称与头像 | **OpenID、授权的头像/昵称、设备识别信息 (用于安全防护)** | [腾讯隐私保护指引](https://privacy.qq.com/)        |
| **腾讯科技**           | **Tinker (热修复)** | 用于修复 App 运行中的紧急 Bug，提升稳定性  | **设备型号、系统版本、网络状态**                         | [Tinker GitHub](https://github.com/Tencent/tinker) |
| **Coil 团队**          | **Coil (图像加载)** | 加载云端题目图片、DiceBear 头像、QQ 头像   | **网络访问权限 (仅为下载并显示图片)**                    | [Coil 官网](https://coil-kt.github.io/coil/)       |
| **DeepeSeek / 合作方** | **AI 分析模块**     | 提供题目讲解、学情分析报告                 | **脱敏后的题目 ID、答题对错情况 (由本平台转发)**         | [DeepSeek 隐私协议](https://www.deepseek.com/)     |

### 特别说明：

1. **设备权限使用：** 我们仅在必要时申请网络访问、存储读写（用于缓存题目图片和本地数据）权限。
2. **数据加密：** 与上述第三方进行数据交互时，均采用 HTTPS 等加密协议，确保数据传输安全。
3. **最小化原则：** 我们仅向第三方提供实现相关功能所必须的最小化信息，绝不主动泄露您的通讯录、精准定位等敏感隐私。
